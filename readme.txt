== Description ==

Um plugin para analisar o servidor de hospedagem.
Ele te mostra quais as configuracoes do seu servidor de hospedagem e te mostra o desempenho do seu site em relacao as configuracoes do servidor.
Ideal para sites que estao travando, hospedagem compartilhada, servidores fracos, e-commerce, sistemas integrados, portais e muitos outros.
Saiba quantos acessos seu servidor suporta.

== Installation ==

1. Upload `server-analytics` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
